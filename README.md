# IVIX_TV site

Статический сайт для Twitch-канала IVIX_TV.

## Локальный запуск

```bash
python3 -m http.server 8000
```

Открыть:

```text
http://localhost:8000
```

## Деплой

Проект настроен для Cloudflare Pages.

Настройки:

```text
Framework preset: None
Build command: пусто
Build output directory: /
```
