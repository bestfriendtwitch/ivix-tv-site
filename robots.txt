# IVIX_TV site

Статический сайт для Twitch-канала IVIX_TV.

## Локальный запуск

```bash
python3 -m http.server 8000
```

Открыть:

```text
http://localhost:8000
```

## Деплой

Проект подключен к Cloudflare Pages через GitHub.

Настройки Cloudflare Pages:

```text
Framework preset: None
Build command: пусто
Build output directory: /
```

## Аналитика

В `index.html` и `404.html` добавлен Cloudflare Web Analytics snippet для текущего hostname `ivix-tv-site.pages.dev`.

Когда основной домен `ivixtv.ru` будет активирован и подключен к Pages, нужно добавить/обновить hostname в Cloudflare Web Analytics и при необходимости заменить snippet/token.
